# Жаңа тақырып жасау — қысқаша нұсқаулық / Creating a new topic

Бұл — **сабақ файлдары** (lesson files): бір тақырыпқа арналған теория + есептер,
әрқайсысы бөлек `.js` файл. GeoGebra емес — қарапайым JavaScript + SVG.

## Жұмыс тәртібі / Workflow

1. **Осы `_template/` папкасын көшіріп**, жаңа атпен қойыңыз
   (мысалы, `5-2-1/` — сынып-тарау-тақырып). `_template`-тің өзін өзгертпеңіз.
2. Файлдарды толтырыңыз (төмендегі кестені қараңыз). **Әр `id`-ді өзгертіңіз** —
   «TEMPLATE-…» орнына тұрақты, қайталанбайтын атау (тек `a-z 0-9 -`).
   `id` — тұрақты бірегейлік: сол `id`-мен қайта жүктесеңіз, орнында жаңарады.
3. **`previewer.html`-ды браузерде ашыңыз** (екі рет басу) → файлдарды сүйреп
   тастаңыз → әр файлды KZ және RU тілінде басып тексеріңіз.
4. Дұрыс болса → **/admin/lessons** → тақырыпты ашып, папканы сүйреп жүктеңіз →
   кестеде difficulty/tags/order қойыңыз.
5. Тақырыпты /grades бетіне шығару: `topics.lesson_topic_id`-ті сол grade
   тақырыбына байлаңыз (админде). **Deploy керек емес — мазмұн = дерек.**

Өзгерткіңіз келсе: файлды түзетіңіз → previewer-де қараңыз → сол `id`-мен
қайта жүктеңіз. Болды.

## Қай файлды қашан / Which file for what

| Copy this        | For                                                        |
| ---------------- | ---------------------------------------------------------- |
| `theory.js`      | Тақырыптың теориясы. **Бір бөлім = бір слайд** (Алдыңғы/Келесі). Әр бөлімде: тақырып + мәтін + (қаласаңыз) сурет + (қаласаңыз) жасырын жауап. |
| `problem-figure.js` | Суреті бар есеп (сан сәулесі, анимация, эмодзи...). Мұғалім «Түсіндіруді көрсету»-ді басқанша шешуі жасырын. |
| `problem-text.js`   | Суретсіз есеп (арифметика, логика). Шарты жоғарыда, шешуі жасырын. |

Бір тақырыпта: **бір `theory.js`** + қалағанша есеп файлы. Есеп файлдарының
атын еркін қойыңыз (`problem1.js`, `chase.js`...) — маңыздысы meta ішіндегі `id`.

## Толық ережелер / Full docs

- **`../FORMAT.md`** — форматтың толық келісімі (esp. §1b «document-mode» +
  «Document-layout theory»). Жаңа файл жазбас бұрын оқыңыз.
- **`../SKILL.md`** — авторлық шеберлік: toolkit, педагогика, тексеру тізімі.
- **`../5-1-2/`** — толық, тірі мысал (теория + №19–27): статик суреттер,
  анимациялы қуалау (chase), анимациялы секірулер, жасырын жауаптар. Ең жақсы
  үлгі — осыны қараңыз.

## Есте сақтаңыз / Gotchas

- Әр файл **IIFE-ге оралады** (`(function () { … })();`) — файлдар бір
  бетте бір scope-та жүреді, helper-лер бір-біріне кедергі жасамасын.
- Барлық көрінетін мәтін `{ kz, ru }`. Математика: `\( … \)` жол ішінде,
  `\[ … \]` бөлек жол.
- Тек мына класстар стильденеді: `lf-formula`, `lf-given`, `lf-answer`,
  `lf-callout`, және `<b class="lf-find">…</b>` (табу керегі).
- Сурет `<svg viewBox="0 0 W H" style="width:100%;height:auto">` ішіне
  салынады — кез келген еніне бейімделеді.
- Таймер/анимация бастаған `visual` **`destroy()` қайтаруы керек**.
